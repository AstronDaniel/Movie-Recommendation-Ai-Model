# Model Training

This folder contains notebooks and scripts for training the movie recommendation model.

## Structure

- `notebooks/` - Jupyter notebooks for model development, training, and evaluation
- `data/` - Training datasets (add your movie datasets here)
- `models/` - Saved trained models (generated after training)

## Usage

1. Add your training data to the `data/` folder
2. Open and run the notebooks in `notebooks/` to train your models
3. Save trained models to the `models/` folder
4. Update `movie_recommender.py` to load trained models
